## 배포

1. git clone
```
git clone https://github.com/woowa-techcamp-2022/web-moneybook-10.git
```

2. install package
```
cd frontend
npm i

cd ../backend
npm i
```

3. set .env
```
# frontend
HOST=

# backend
HOST=
PORT=
DATABASE_HOST=
DATABASE_USERNAME=
DATABASE_PASSWORD=
DATABASE_PORT=
DATABASE_NAME=
```
